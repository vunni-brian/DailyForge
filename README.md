# DailyForge

DailyForge is a desktop-first productivity workspace built with React, TypeScript, and Vite.

Current app scope:

- dashboard
- tasks with kanban and list views
- projects workspace
- notes editor
- focus timer
- learning tracker
- daily reviews
- settings with local export/import

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
